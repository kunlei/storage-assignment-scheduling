To run this program, it's required to set up a virtual environment using `conda` first:

```commandline
conda create --name mujin-env --file requirements.txt
```

Once the virtual environment is ready, activate it using

```commandline
conda activate mujin-env
```

Now navigate to the directory where the repository is saved, run the following command to test:

```commandline
python main.py
```
